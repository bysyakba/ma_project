При отправке какого-то числа по адресу "http://любойадресзаглушка" - нам возвращается указанное количество емейлов
Ошибка 404 возвращается, если:
    количество емейлов меньше или равно - 2
    количество емейлов больше или равно - 11
    передано не число
Если всё ок - возвращается список емейлов в количестве указанного числа


емеилы генерируются через Faker
